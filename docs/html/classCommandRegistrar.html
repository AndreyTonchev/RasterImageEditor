Detailed Description

template<typename T>
class CommandRegistrar< T >

Template helper for registering commands with the factory.

Each specific command class T must inherit from Command and provide a constructor taking Session* as its only argument.

Registration is performed automatically via static initialization.

TThe command type to register.
