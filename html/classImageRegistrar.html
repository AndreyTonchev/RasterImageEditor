Detailed Description

template<typename T>
class ImageRegistrar< T >

Template class to register image format handlers with the ImageFactory.

This class acts as a registrar that automatically registers a given image type with the factory upon construction. It inherits from ImageBase and provides a create method to instantiate images of the registered type.

Template Parameters
TThe image class type that must have a constructor accepting a filename.
