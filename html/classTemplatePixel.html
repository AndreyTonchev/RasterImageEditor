Detailed Description

template<typename DataType, std::size_t Channels>
class TemplatePixel< DataType, Channels >

A templated pixel implementation supporting arbitrary data types and number of channels.

Template Parameters
DataTypeThe underlying data type for each channel (e.g., uint8_t, uint16_t).
ChannelsThe number of channels (e.g., 1 for grayscale, 3 for RGB).
