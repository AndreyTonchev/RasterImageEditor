Detailed Description

Declares the PGMImage class for handling PGM image files (P2 ASCII and P5 binary formats).

The PGM image format stores grayscale pixel data. Supported formats:

  • P2: ASCII encoding of pixels.
  • P5: Binary encoding of pixels.

File structure:

  • Signature (P2 or P5)
  • Width (unsigned int)
  • Height (unsigned int)
  • max_value (maximum pixel value)
  • List of pixels (grayscale values ranging from 0 (black) to max_value (white))
